# Explorando dados com Excel

## Abrindo a base no Excel

1. Abra o arquivo `clientes_dataset.xlsx`
2. Observe:

- Quantas colunas existem?
- O que cada coluna representa?
- Existem células vazias? Valores estranhos?

> [!TIP]
> Dica: use o botão de 'Congelar Painéis' para ficar o cabeçalho

## Tipos de dados

O Excel pode armazenar vários tipos de dados:

- Texto (string): nomes, categorias
- Números: idade, valor, quantidade
- Datas: nascimento, data de compra
- Valores booleanos: sim/não, verdadeiro/falso

## Aplicando filtros e ordenação

Você pode usar:

- Filtros automáticos para visualizar somente o que interessa
- Ordenação crescente/decrescente para detectar padrões

Exemplos:

- Quais clientes têm mais de 50 anos?
- Quais são os 10 maiores gastos?

## Exemplo prático - Responder perguntas com Excel

Com a base de clientes, tente responder:

- Qual a média de idade dos clientes?
- Quantos clientes estão no estado de SP?
- Quem é o cliente com maior valor gasto?
- Qual o gênero mais comum?

Use filtros, fórmulas como `=MÉDIA()` e `=CONT.SE()`

## Dica bônuns - Tabelas dinâmicas

Com tabelas dinâmicas, você pode:

- Resumir dados por categoria
- Criar contagens, somas, médias por grupo
- Comparar comportamentos

Exemplo: total gasto por gênero ou por faixa etária

> [!TIP]
> Menu -> Inserir -> Tabela Dinâmica
