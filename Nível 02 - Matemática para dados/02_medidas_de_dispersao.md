Exercício 3 - Resposta:

```
  x = 12 - 8
  x = 4
```

Exercício 4 - Resposta:

```
  x = (10 + 12 + 8 + 10 + 10) / 5
  x = 50 / 5
  x = 10
```

Exercício 5 - Resposta:

```
  (10 - 10) ** 2 = 0
  (12 - 10) ** 2 = 4
  (8 - 10) ** 2 = 4
  (10 - 10) ** 2 = 0
  (10 - 10) ** 2 = 0

  x = 0 + 4 + 4 + 0 + 0 / 5
  x = 8 / 5
  x = 1,6
```
